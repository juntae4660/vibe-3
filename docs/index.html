PRD 기능 요구서

이 슈퍼앱은 공공직군 행정 실무자가 반복 수행하는 일정 공유, 엑셀 정리, 민원 응대, 행정 뉴스 확인 업무를 하나의 웹 애플리케이션에서 처리하도록 돕는 플랫폼입니다.

제품 목표

  • 팀원의 휴가, 근무, 출장 일정을 한 화면에서 공유합니다.
  • 엑셀 파일을 기준 컬럼에 따라 자동 분리하거나 여러 파일을 병합합니다.
  • 민원 매뉴얼을 기반으로 대응 방식과 답변 스크립트 초안을 생성합니다.
  • 공공 행정 관련 뉴스 기사를 매일 아침 수집하고 요약 확인합니다.
  • 비개발자 행정 담당자도 브라우저에서 사용할 수 있게 합니다.

MVP 범위

스케줄 CRUD 엑셀 업로드 컬럼별 분리 파일 병합 FastAPI SQLite

챗봇 LLM 연동과 뉴스 정기 수집 자동화는 MVP 이후 고도화 대상으로 두고, 초기에는 매뉴얼 업로드/검색과 뉴스 수동 수집부터 시작하는 것이 현실적입니다.

핵심 기능

팀원 스케줄 관리 휴가, 근무, 출장, 외근, 교육 일정을 캘린더에서 공유하고 팀원/유형별로 필터링합니다.
엑셀 업무 자동화 엑셀 파일을 업로드하고 특정 컬럼 기준으로 분리하거나 여러 파일을 병합합니다.
민원 대응 챗봇 첨부된 민원 매뉴얼을 근거로 대응 절차와 답변 스크립트 초안을 제공합니다.
뉴스 기사 수집 공공 행정 관련 뉴스를 키워드 기반으로 수집하고 제목, 출처, 링크를 저장합니다.

API 초안

  • GET /api/calendar/events: 일정 목록 조회
  • POST /api/calendar/events: 일정 생성
  • POST /api/excel/analyze: 엑셀 컬럼 분석
  • POST /api/excel/split: 기준 컬럼별 엑셀 분리
  • POST /api/excel/merge: 여러 엑셀 파일 병합
  • POST /api/chatbot/respond: 민원 대응 초안 생성
  • GET /api/news: 뉴스 목록 조회
  • POST /api/news/collect: 뉴스 수집 수동 실행

Operation

현재 상태

  • FE 필수 모듈은 frontend/에 설치되어 있습니다.
  • BE는 backend/.venv 가상환경과 fastapi가 설치되어 있습니다.
  • 실제 애플리케이션 코드는 아직 작성 전입니다.
  • SQLite는 Python 내장 sqlite3로 사용 가능합니다.

확인 명령

cd frontend
npm.cmd list --depth=0
cd backend
$env:UV_CACHE_DIR='.uv-cache'
uv run python -c "import fastapi; print(fastapi.__version__)"

실행 예정 명령

cd frontend
npm.cmd run dev
cd backend
$env:UV_CACHE_DIR='.uv-cache'
uv run fastapi dev app/main.py

자주 발생하는 오류

PowerShell npm 실행 정책 오류

npm.ps1 cannot be loaded 오류가 발생하면 npm 대신 npm.cmd를 사용합니다.

uv 캐시 접근 권한 오류

Failed to initialize cache 오류가 발생하면 backend에서 $env:UV_CACHE_DIR='.uv-cache'를 지정한 뒤 uv 명령을 실행합니다.

python 명령 없음

시스템 Python 대신 uv 관리 Python을 사용합니다. uv run python --version으로 확인합니다.

sqlite3 CLI 없음

SQLite CLI가 없어도 Python 내장 sqlite3로 개발할 수 있습니다. DB 직접 확인이 필요하면 DB Browser for SQLite 같은 도구를 사용합니다.
